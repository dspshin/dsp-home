1. 타워디펜스
- 링크 https://tower.dsp.ai.kr/
- 아이콘링크 https://tower.dsp.ai.kr/icon.png
- **Merge Defense**는 Next.js와 FastAPI를 사용하여 구축된 최신 풀스택 아키텍처 기반의 웹 타워 디펜스 게임입니다. 플레이어는 타워를 소환하고, 머지(병합)하여 업그레이드하며, 적의 웨이브를 막아내야 합니다. 이 게임은 클라우드 저장, 안전한 백엔드 검증, 오프라인 보상 기능을 제공합니다. 다만 작은 모바일 화면에의 최적화는 되어 있지 않으므로 노트북이나 데스크탑, 테블릿으로 실행하는 것이 좋습니다.
- 기술 스택
-- **Frontend**: Next.js 14+ (App Router), TypeScript, HTML5 Canvas
-- **Backend**: FastAPI, Python 3.9+, Pydantic
-- **Database**: Supabase (PostgreSQL)

2. 2026 병오년(붉은 말의 해) 신년 운세 웹 앱
- 링크 https://saju.dsp.ai.kr/
- 아이콘링크 https://saju.dsp.ai.kr/icon.png
- **정밀한 만세력**: `lunar-javascript` 라이브러리를 활용하여 사용자의 생년월일시를 기반으로 60갑자(Ganji)를 정확하게 산출합니다.
- **2026년 운세 해석**: 사용자의 일간(Day Master)과 오행(Five Elements)을 분석하여 2026년 병오년의 불(Fire) 기운과의 조화를 해석합니다.
- **다양한 운세 항목**: 총운, 재물운, 애정운, 건강운, 직장/사업운, 그리고 행운의 아이템/색상까지 제공합니다.
- 기술 스택 (Tech Stack)
-- **Framework**: [Next.js 15+ (App Router)](https://nextjs.org/)
-- **Language**: TypeScript
-- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
-- **Icons**: Lucide React
-- **Logic**: [lunar-javascript](https://github.com/6tail/lunar-javascript) (사주/만세력 계산)
